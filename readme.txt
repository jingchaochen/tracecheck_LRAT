Verifying traceCheck dependency graphs in LRAT: tracecheck_LRAT
Authors : Jingchao Chen

tracecheck_LRAT can be built with g++ command:
./build.sh

For running
cd bin 
./tracecheck_LRAT <trace-file-in-LRAT> 

For detailed LRAT formats, see the following paper.
